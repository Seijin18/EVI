# Convenções de tools

- **summarize_inbox** — retorna `messages[]` com id, subject, from.
- **delete_emails** — batch por IDs (do snapshot ou listagem).
- **delete_emails_by_query** — Gmail search, ex. `from:aliexpress OR from:olx`.
- **confirm_commitments / dismiss_commitments** — ids numéricos da fila pendente.
